Welcome

Hello, I've been making games at some capacity for over five years. I have taught myself a number of programs using online tutorials and other resources to assist me. I make games using the Unity game engine, using the C# programming language for scripting. I have also taught myself some more general digital skills such as modelling in blender, basic video editing in Adobe Premiere Pro, and basic image editing in Adobe Photoshop.

This website acts as an online portfolio which demonstrates the kind of work that I am capable of producing. Much of my work focuses on surreal or intimidating creatures and environments.
